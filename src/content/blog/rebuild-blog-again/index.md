## Hi everyone,

After a long time Im back, aaand yes I rebuilt the blog again. This time with [Astro](https://astro.build/).

The DX was honestly super pleasant and the migration was pretty straightforward since this is a pretty small project. You might be wondering why I bothered… well, the old version was starting to feel a bit messy mmmmm not bit but a lot heehee, so I decided to tidy everything up while I was at it.

I also wanted to try Astro, heard so many good things about it and that went great. Vibe coding helped a lot in this redesign. I was genuinely impressed by how well [Claude Opus](https://www.anthropic.com/claude/opus) performed. I managed to get like 80% of it done before my credits ran out. The rest I tinkered myself and [Gemini](https://deepmind.google/models/gemini/pro/) did some heavy lifting.

And yes this was like the first time I tried [Agentic coding](https://cloud.google.com/discover/what-is-agentic-coding) which fully goes from planning to execution phase, because I didnt get any proper project to try it out. I used to not be a big fan of [Agentic AI](https://cloud.google.com/discover/what-is-agentic-ai) for a while, but with the way things went, it feels like its onto something big. Pretty soon its going to be an integral part of how we work, which we might not be able to get any work done without it.

Like today, I was in the middle of something, credits burned out, and I honestly considered paying for the model just for the convenience. 

And I was like wait.. how did I even used to do this before? Its a little scary how fast things have changed.

Ol days we used to move laterally across millions of tabs on [stackoverflow](https://stackoverflow.com/) and multiple stack exchange sites in debugging hell and now, ask from [grok](https://grok.com/), [claude](https://claude.ai/), [gemini](https://gemini.google.com/) and will probably solve 80% of the issue. I mean Iam not saying its bad or perfect, but it feels different and pretty easy to get distracted from the core problem if we arent vigilant.

Thats it for this post. Hoping to drop a proper one soon.
